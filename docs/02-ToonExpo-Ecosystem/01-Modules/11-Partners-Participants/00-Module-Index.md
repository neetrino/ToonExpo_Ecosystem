# Partners / Participants - Module Index

## Purpose

Partners / Participants represents non-builder companies participating in ToonExpo.

Examples:

- banks;
- IT companies;
- sponsors;
- suppliers;
- insurance;
- legal;
- design/furniture;
- service companies;
- other event participants.

## Core Rule

Banks are a partner type with special bank offer fields.

The partner module should not be designed only around banks.
