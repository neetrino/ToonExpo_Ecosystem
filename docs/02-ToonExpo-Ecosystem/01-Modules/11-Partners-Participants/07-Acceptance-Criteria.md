# Acceptance Criteria

## Partner Profiles

- BigProjects Admin can create partner company.
- BigProjects Admin can set partner type.
- BigProjects Admin can edit partner profile.
- Partner profile can be draft/published/archived.
- Published partner appears on public partner list.
- Partner detail page shows public profile data.

## Partner Access

- Partner does not self-register publicly.
- Partner account can be created by BigProjects Admin/staff.
- Partner account can be created through BOS provisioning if integrated.
- Partner can edit own profile only if enabled.
- Partner cannot edit other partner/builder data.

## Bank Extension

- Bank is represented as partner type `bank`.
- Bank partner can have bank offer records.
- Bank offer can appear on mortgage page if published.
- Bank partner profile remains part of partner module.

## Boundaries

- Partner module does not calculate mortgage payments.
- Partner module does not manage builder CRM.
- Partner module does not manage builder project inventory.
- Service Provider Directory remains separate from general partners unless intentionally linked later.

## Out Of Scope Confirmation

v1 does not require:

- partner self-registration;
- partner readiness;
- complex partner portal;
- bank API integration;
- mortgage application flow.
