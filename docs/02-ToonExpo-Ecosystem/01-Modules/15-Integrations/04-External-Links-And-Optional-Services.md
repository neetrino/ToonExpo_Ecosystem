# External Links And Optional Services

## Matterport / 3D Links

Apartment can store external 3D/Matterport links.

v1 can:

- show link/button;
- optionally embed if simple and reliable.

No deep integration is required.

## Map Services

External map service may be used for:

- public project location;
- address/location display.

Exhibition venue routing does not require live GPS or external indoor positioning in v1.

## Email / Invitation

ToonExpo can send invitation/login emails after admin/BOS-created accounts.

Exact provider and email implementation are technical decisions.

## Out Of Scope External Integrations

v1 does not require:

- bank API integration;
- external CRM sync;
- payment gateway;
- ticketing provider;
- marketing automation;
- document-signing workflow.
