# Admin / Content Management - Module Index

## Purpose

Admin / Content Management gives BigProjects control over ToonExpo content, companies, public presentation and platform settings.

It is the operational admin layer for ToonExpo.

## Core Rule

v1 is not a heavy moderation system.

Use simple publication status:

```text
draft
published
archived
```
