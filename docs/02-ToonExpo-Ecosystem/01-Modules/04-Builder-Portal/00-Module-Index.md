# Builder Portal - Module Index

## Purpose

Builder Portal is the private ToonExpo workspace for approved builder participants.

It gives builders one place to manage:

- company profile and public presentation;
- projects, buildings, floors and apartments;
- media and visual map setup;
- buyer requests and Constructor CRM;
- readiness checklist/evaluation;
- exhibition map placement visibility;
- analytics related to their own activity.

## Core v1 Rules

- Builder accounts are created by BigProjects Admin/staff or through BOS provisioning.
- Builders do not self-register.
- Builder Portal is not a separate product from ToonExpo; it is a role-based private area inside the ToonExpo ecosystem.
- In v1, builder-side roles stay simple. All builder company members can use the same company workspace unless a later phase adds deeper subroles.
- Details of one entity open in a side sheet from the current workspace.
- Files/media are attached to real entities; there is no separate generic Files/Documents module in v1.
- BigProjects Admin can manage all builder data inside ToonExpo, especially during the early phase when admins fill data for builders.
