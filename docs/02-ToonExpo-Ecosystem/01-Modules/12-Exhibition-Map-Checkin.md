# Public Exhibition Map

## Status

Canonical map module. The existing folder name is retained for link stability, but check-in is not owned by this module and is outside the current map delivery.

## Purpose

ToonExpo stores and renders an immutable public copy of the venue map published by BigProjects BOS.

Release scope includes map browse, search, area selection, public participant information, landmarks and an optional device-local approximate `I am here` marker. Professional path generation is deferred.
