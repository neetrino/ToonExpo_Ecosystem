# Visual Map / Hotspots - Module Index

## Purpose

This folder contains deep documentation for visual navigation inside real estate projects.

This module lets users move through prepared 2D visuals:

```text
Project visual
-> Building hotspot
-> Building / floor selection
-> Floorplan visual
-> Apartment hotspot
-> Apartment page
```

## v1 Principle

v1 is not a full map/CAD editor.

v1 uses ready images from builders or BigProjects and places clickable markers on top.

Start simple:

- one visual image per context when needed;
- point markers;
- target links to building/floor/apartment;
- public fallback lists when visuals are incomplete.

## Not The Same As

This module is not Exhibition Map & Check-in.

Exhibition Map & Check-in is for the physical event venue, booths, entrance and route/path inside the pavilion.

Visual Map / Hotspots is for navigating real estate project visuals.
