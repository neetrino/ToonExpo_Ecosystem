# BigProjects Analytics

## Purpose

BigProjects Admin needs global visibility across ToonExpo.

## Dashboard Sections

Recommended v1 sections:

- platform activity;
- project/apartment performance;
- request/CRM summary;
- QR/check-in summary;
- readiness summary;
- partner/bank summary;
- event map/booth activity.

## Platform Activity

Show:

- total users;
- buyers registered;
- builders active;
- partners active;
- projects published;
- apartments published;
- public page views.

## Real Estate Performance

Show:

- top projects by views;
- top apartments by views;
- favorites count;
- requests count;
- available/reserved/sold summary.

## CRM Summary

Show:

- requests created;
- deals by status;
- source breakdown;
- builder-level summaries.

BigProjects Admin should view analytics by default, not edit builder sales CRM.

## QR / Check-in Summary

Show:

- QR scans by context;
- check-ins;
- duplicate check-ins;
- invalid/blocked scans;
- event attendance over time.

## Readiness Summary

Show:

- builders ready;
- builders needing improvement;
- blocked categories;
- common weak categories.

## Export

Optional v1:

- CSV export for admin summary tables.

Full reporting/BI can be v2.
