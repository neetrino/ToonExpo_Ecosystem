# Acceptance Criteria

## Tracking

- System can record project views.
- System can record apartment views.
- System can record favorites.
- System can record request creation.
- System can record QR scans.
- System can record check-ins.
- System can record booth selections and route requests.

## BigProjects Analytics

- BigProjects Admin can view global analytics.
- BigProjects Admin can filter by event/time where applicable.
- BigProjects Admin can view request/source summary.
- BigProjects Admin can view check-in summary.
- BigProjects Admin can view readiness summary.

## Builder Analytics

- Builder can view own analytics only.
- Builder can see own project/apartment views.
- Builder can see own requests/CRM summary.
- Builder cannot see other builder analytics.

## Boundaries

- Analytics does not edit CRM data.
- Analytics does not calculate readiness scores.
- Analytics does not validate check-in.
- Analytics does not include paid ticket/payment reports in v1.
- Full BI warehouse is out of scope for v1.
