# Builder Analytics

## Purpose

Builder should see own company/project performance.

## Builder Visibility

Builder can see only own data:

- own profile views;
- own project/apartment views;
- own favorites;
- own requests;
- own CRM summary;
- own readiness progress.

Builder cannot see:

- other builders' analytics;
- global platform analytics;
- buyer personal data outside own CRM interactions.

## Dashboard Sections

Recommended:

- project views;
- apartment views;
- favorites;
- requests;
- CRM status summary;
- apartment status summary;
- readiness status.

## CRM Summary

Show:

- new requests;
- contacted/follow-up needed;
- reserved;
- converted;
- lost/closed.

## Apartment Summary

Show:

- available;
- reserved;
- sold;
- most viewed apartments;
- apartments with requests.

## Time Filters

Recommended v1 filters:

- today;
- last 7 days;
- last 30 days;
- event period if active;
- custom later.
