# SEO I18n And Performance

## SEO

Public pages should support basic SEO:

- title;
- description;
- clean URL/slug;
- social preview image if useful;
- indexable public project/apartment/partner pages where appropriate.

## Languages

Supported languages:

- Armenian;
- Russian;
- English.

Public content should support translations.

## Fallback Language

If translation is missing in v1, platform can fall back to default language.

This should be visible/understandable, not broken empty content.

## Performance

Public pages are image-heavy.

v1 should:

- optimize images;
- lazy load galleries;
- avoid loading huge original floorplans by default;
- keep mobile page load fast;
- cache public content where safe.

## Accessibility

Provide:

- semantic page structure;
- alt text where available;
- keyboard reachable controls;
- text fallback for visual maps;
- color labels not relying only on color.
