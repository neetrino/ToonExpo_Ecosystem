# Public Web / Mobile App - Module Index

## Purpose

Public Web / Mobile App is the public face of ToonExpo on `toonexpo.com`.

It lets visitors browse projects, apartments, builders, partners, mortgage offers, service providers and exhibition map.

## Core Rule

Mobile experience is a primary product surface.

The public web should feel app-like on mobile because a large share of users is expected to use phones.
