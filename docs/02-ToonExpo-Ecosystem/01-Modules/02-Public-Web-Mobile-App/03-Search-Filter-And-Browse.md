# Search Filter And Browse

## Search Scope

Public search can include:

- projects;
- apartments;
- builders;
- partners;
- banks;
- exhibition booths/projects during event.

## Project Filters

Recommended v1:

- location;
- builder;
- status;
- availability;
- price range if public;
- rooms/area if apartment search is available.

## Apartment Search

Apartment search/list is useful but project-first browsing remains primary.

Apartment filters:

- project;
- building;
- floor;
- rooms;
- area;
- price visibility/price;
- status.

## Map/List Views

Public pages can use:

- card/list view;
- visual map/hotspot navigation;
- exhibition map for event context.

Do not confuse project visual map with exhibition venue map.

## Empty States

Show useful empty states:

- no projects found;
- no apartments matching filters;
- no published bank offers;
- no providers in category;
- exhibition map not published yet.

## Search Result Actions

Actions can include:

- open detail page;
- save/favorite;
- request contact/price;
- show on exhibition map if event booth exists.
