# Public Pages And Navigation

## Public Sitemap

Core public pages:

- Home;
- Projects;
- Project Detail;
- Building Detail;
- Floor View;
- Apartment Detail;
- Builders;
- Builder Detail;
- Partners;
- Partner Detail;
- Mortgage;
- Service Provider Directory if enabled;
- Exhibition Map;
- Favorites;
- Profile / My QR.

## Home

Home should provide quick access to:

- projects;
- apartments/search;
- builders;
- exhibition map;
- mortgage;
- buyer QR/profile after login.

## Projects

Projects page lists published projects.

Filters can include:

- builder;
- location;
- project status;
- apartment availability;
- price range if public.

## Project Detail

Project detail uses data from Projects module:

- project overview;
- buildings;
- visual navigation if available;
- apartment summary;
- request CTA.

## Apartment Detail

Apartment detail should show:

- apartment data;
- status;
- price visibility;
- plan/gallery;
- Matterport/3D link if available;
- request CTA;
- save/favorite action.

## Builders

Builder list/detail should show published builder profiles and projects.

## Partners

Partner list/detail should show published partner profiles.

Bank partners can link to mortgage offer.

## Mortgage

Mortgage page shows bank partner offers and calculator.

## Exhibition Map

Exhibition map page shows physical event venue map, booth search and route/path.

## Authenticated Shortcuts

When logged in as buyer:

- Favorites;
- Requests;
- My QR;
- Profile.
