# Buyer / Visitor Area - Module Index

## Purpose

Buyer / Visitor Area is the personal area for registered ToonExpo visitors.

It includes:

- profile;
- My QR;
- favorites/saved items;
- requests and interest history;
- check-in identity/status.

## Core Rule

Buyer-facing history is not builder CRM.

Builder sees CRM deal/request.

Buyer sees request/interest/contact history.
