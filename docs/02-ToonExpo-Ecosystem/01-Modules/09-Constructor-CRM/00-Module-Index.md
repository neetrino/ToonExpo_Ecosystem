# Constructor CRM - Module Index

## Purpose

Constructor CRM is the builder sales workspace inside ToonExpo.

It receives requests from CRM Lead Intake and lets builders manage buyer follow-up, apartment selection, reservation and sale.

## Core Rule

Buyer requests from public pages and builder-created QR requests must appear in the same CRM pipeline.

Do not create a separate Requests / Leads workspace outside CRM for builders.

## v1 Principle

Keep CRM operational and understandable:

- one pipeline;
- one deal/request sheet;
- simple statuses;
- apartment link when needed;
- activities/notes inside deal;
- no separate general task management system.
