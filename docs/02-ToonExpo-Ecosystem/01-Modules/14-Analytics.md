# ToonExpo Module: Analytics

## Status

v1 basic

## Purpose

Analytics helps BigProjects and builders understand activity inside ToonExpo.

## In Scope

- project views;
- apartment views;
- favorites;
- requests/leads;
- QR scans;
- check-ins;
- builder CRM summary;
- readiness progress;
- public profile completion.

## Audience

- BigProjects admin;
- builder company users for own data;
- optional admin exports/reports;
- account provisioning status if needed for BOS integration.

## Out Of Scope

- full BI warehouse in MVP;
- advanced attribution model;
- paid ticket/payment reports.
