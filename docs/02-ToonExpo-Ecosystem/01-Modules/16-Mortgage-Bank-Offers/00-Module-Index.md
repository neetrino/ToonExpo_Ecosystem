# Mortgage / Bank Offers - Module Index

## Purpose

Mortgage / Bank Offers is a public page that lets buyers compare bank partner offers with a simple calculator.

Route:

```text
/mortgage
```

## Core Rule

This module is informational/calculator-only in v1.

It does not submit mortgage applications.
