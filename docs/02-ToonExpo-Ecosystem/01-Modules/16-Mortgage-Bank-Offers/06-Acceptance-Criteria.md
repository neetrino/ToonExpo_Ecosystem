# Acceptance Criteria

## Public Page

- Mortgage page is available at `/mortgage`.
- Published bank offers appear on mortgage page.
- Draft/archived bank offers do not appear publicly.
- Buyer can adjust property price.
- Buyer can adjust down payment.
- Buyer can select loan term.
- Buyer can select bank offer.
- Monthly estimate changes when inputs or selected bank change.

## Bank Offers

- BigProjects Admin can create bank offer for bank partner.
- BigProjects Admin can edit/publish/archive bank offer.
- Bank partner can edit own offer only if enabled.
- Bank offer requires rate, min down payment and term option.
- Bank offer must belong to partner type `bank`.

## Boundaries

- No mortgage application is submitted in v1.
- No document collection exists in v1.
- No bank API integration exists in v1.
- No payment flow exists in v1.
- Calculator result is an estimate, not approval.

## Mobile

- Mortgage page works on mobile.
- Offer cards are readable on mobile.
- Calculator inputs are usable on mobile.
