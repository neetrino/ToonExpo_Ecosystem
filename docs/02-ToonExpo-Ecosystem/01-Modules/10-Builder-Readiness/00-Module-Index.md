# Builder Readiness - Module Index

## Purpose

Builder Readiness helps BigProjects evaluate how ready a builder/project is for ToonExpo public presentation, event participation and buyer interaction.

It turns quality checks into:

- scores/statuses;
- recommendations;
- required actions;
- builder-facing guidance;
- service provider suggestions.

## v1 Principle

Readiness should be useful before it is sophisticated.

v1 can be manual or semi-automatic:

- BigProjects evaluates;
- system shows scores/statuses;
- builder sees what to improve;
- weak categories can show relevant service providers.

Do not block v1 on complex automatic scoring.
