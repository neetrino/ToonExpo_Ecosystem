# ToonExpo Module: Public Web / Mobile App

## Status

v1

## Purpose

Public Web / Mobile App is the public face of ToonExpo.

## In Scope

- public website on `toonexpo.com`;
- mobile-first UX;
- app-like mobile web experience;
- app publication path via Neetrino App Technology;
- project catalog;
- builder pages;
- apartment pages;
- partner/participant pages;
- mortgage / bank offers page;
- service provider directory page if enabled;
- search/filter;
- map/list views;
- request CTAs.

## Out Of Scope

- paid tickets;
- payment flow;
- push notifications in MVP;
- online mortgage application flow.

## UX Rule

Mobile experience is critical. Around 60% of usage is expected to be mobile.
