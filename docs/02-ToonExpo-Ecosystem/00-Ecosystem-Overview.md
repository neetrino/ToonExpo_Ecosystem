# ToonExpo Ecosystem Overview

## Definition

ToonExpo Ecosystem is one ToonExpo platform/ecosystem for public users, buyers, builders, participants, CRM, readiness, QR and event experience.

It is not three separate platforms for the user.

## Main Rule

ToonExpo should feel like one ecosystem to users, especially builders.

Internally, modules must still have clean boundaries so CRM, readiness, public content and QR logic do not become one messy admin panel.
