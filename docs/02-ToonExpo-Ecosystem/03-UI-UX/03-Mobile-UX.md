# ToonExpo Mobile UX

## Purpose

ToonExpo should be mobile-first. The mobile web/app experience is a core product surface.

## Main Mobile Areas

- project browsing;
- apartment browsing;
- favorites;
- buyer QR;
- QR scanner;
- requests;
- exhibition map/check-in;
- exhibition booth/project search;
- route/path to selected booth/project inside venue;
- builder scan buyer QR action page.

## UX Rules

- app-like navigation;
- large tap targets;
- fast access to QR;
- fast access to favorites;
- search/filter should be mobile-friendly;
- apartment pages should be readable on phone;
- visual map/hotspots must work on mobile.

## Out Of Scope For MVP

- push notifications;
- complex offline mode;
- paid ticket wallet.
