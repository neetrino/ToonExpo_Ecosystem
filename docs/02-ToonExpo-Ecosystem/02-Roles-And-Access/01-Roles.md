# ToonExpo Roles And Access

## Status

Draft

## v1 Main Roles

- Buyer / Visitor;
- Builder;
- Partner;
- BigProjects Admin;
- Entrance Staff.

## Later Detailed Roles

Add only when operationally needed:

- Builder Owner/Admin;
- Builder Sales Manager;
- Builder Viewer;
- Partner Owner/Admin/Editor;
- BigProjects Super Admin;
- BigProjects Platform Admin;
- BigProjects Content Manager;
- BigProjects Readiness Evaluator.

## Key Access Rules

- Buyer has public/buyer area access only.
- Builder sees own company/projects/CRM/readiness.
- Partner sees own partner profile/offers.
- BigProjects can manage ToonExpo content.
- BigProjects has view/analytics access to Constructor CRM by default.
- Entrance Staff sees only scanner/check-in interface.

## v1 Rule

Do not implement detailed sub-roles in v1 unless required.

Use module access and ownership checks to keep access simple.
