# ToonExpo Ecosystem Brief

## Product

ToonExpo Ecosystem is the public, mobile-like, builder, CRM, readiness and event platform for ToonExpo.

It serves buyers/visitors, builders, partners/banks, BigProjects admins and entrance staff.

## Repository Boundary

This repository is only for ToonExpo Ecosystem.

Do not implement BigProjects BOS internal CRM, tasks, KPI, deal onboarding checklist or BOS reports here.

## Users

- BigProjects Admin;
- Builder;
- Partner / Bank;
- Buyer / Visitor;
- Entrance Staff.

## Production Scope

ToonExpo is a full production product, not an MVP or prototype. The list below defines the initial production release, not a reduced-quality implementation.

- account/access foundation;
- public website and mobile-like UX;
- buyer registration and permanent QR;
- builder portal;
- projects/buildings/floors/apartments;
- visual map/hotspots;
- CRM lead intake and Constructor CRM;
- readiness;
- partners and bank offers;
- mortgage calculator page;
- service provider directory;
- public exhibition map synchronized from BOS;
- analytics summaries;
- BOS account provisioning and venue-map publication integrations.

## Project Size

Size C — large, monorepo layout.

Rationale: ToonExpo has many user roles, public/private areas, product inventory, CRM, QR, maps, readiness, partner flows, analytics and external BOS integration.

## Languages

v1 languages:

- Armenian;
